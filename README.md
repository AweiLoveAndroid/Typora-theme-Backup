##  导入方式

>  1、请进入Typora在C盘的缓存目录**C:\Users\Administrator\AppData\Roaming\Typora**，其中`Administrator`是电脑名称，请替换成你的电脑名称。如果你的电脑名称是`zhangsan`，你就换成`zhangsan`。

>  2、然后把本库里面的themes文件夹覆盖上一步打开的路径里面的themes文件夹即可。
